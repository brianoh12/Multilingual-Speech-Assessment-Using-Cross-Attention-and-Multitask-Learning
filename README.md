# Multilingual Speech Assessment Using Cross-Attention and Multitask Learning

This repository is a **paper summary repository** for an Interspeech 2025 publication.

## Overview
This work proposes a multilingual speech assessment approach for non-native spontaneous speech,
covering English, German, and French speech produced by Korean learners.
The method leverages cross-attention and multitask learning to capture both language-specific
and cross-linguistic patterns directly from raw speech.
